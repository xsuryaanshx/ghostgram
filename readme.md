# 👻 GhostGram

> Generate viral influencers that don’t exist.

GhostGram is a fun, interactive web app that creates fake influencer profiles with a swipe-based UI — inspired by Instagram and Tinder.

---

## 🚀 Live Demo
👉 https://your-username.github.io/ghostgram

---

## ✨ Features

- 👤 Random influencer generation (name, bio, followers)
- 📸 Auto-generated aesthetic profile images
- 👉 Swipe left/right (Tinder-style interaction)
- ❤️ Like system with animations
- 💾 Save profiles (stored in browser)
- 🔗 Share profile (copy to clipboard)
- 📂 Saved profiles gallery
- ⚡ Smooth animations & premium UI

---

## 🧠 Concept

GhostGram simulates an AI-powered influencer generator — but runs **100% free with no APIs**, using smart randomization and free image sources.

The goal was to build a **product-like experience**, not just a simple project.

---

## 🛠️ Tech Stack

- HTML
- CSS (Glassmorphism + animations)
- JavaScript (Vanilla)
- LocalStorage (for persistence)
